Trust lands in the first 30 seconds

Three ambient surfaces carry the trust contract of "Curation, in context": the first-run welcome modal that sets expectations before a single tag is curated, the persistent state banner that makes "you are in a non-default state" impossible to miss, and the status bar item that gives an at-a-glance read and a one-click way back in. The copy here is deliberately honest. It promises only what v1.0 ships, and it speaks of optional companions in the conditional, so the welcome modal never claims an integration that is not there.

2 · State banner variants D-007

The state banner docks at the top of every Tag Curator surface whenever the plugin is in a non-default state, so "what state am I in?" is never a question. It has two variants. Amber announces preview mode is on and offers to turn it off; muted announces the plugin is off and offers to turn it on. Each is shown here docked above a curation workspace leaf, exactly where it would appear in Obsidian.

3 · Status bar states D-007

The status bar item is the always-on read of where things stand and the quickest way back into the work. Three states cover the whole picture: curating, with a live count of hidden tags; previewing, with a count of flagged tags; and off. Clicking it opens the curation workspace, pre-filtered to whatever the count refers to.

D-007
The status bar mirrors the state banner so the two never disagree: accent dot while curating, amber dot in preview, faint dot when off. The label always names the current state and the count it reports.
