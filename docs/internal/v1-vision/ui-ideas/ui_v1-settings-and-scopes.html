Scopes D-014

A scope is a display surface where tags render and where Tag Curator can act (vision Section 5.1). v1.0 ships four, each an independent observer on the proven ObserverBase. Every scope is a single toggle row with a one-line description and its own kill switch: turning one off never touches the others, so a misbehaving surface never forces you to disable the whole plugin (principle 11). The riskier surfaces (autocomplete, properties) ship on but are first to flip off if their DOM path shifts across an Obsidian release.
