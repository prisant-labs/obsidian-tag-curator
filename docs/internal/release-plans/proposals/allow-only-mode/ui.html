Allow-only mode

v0.1 ships default mode: rules name tags to hide; unmatched tags stay visible. Allow-only inverts that: only tags matching at least one allow rule are visible; everything else is hidden by default. For users with a curated taxonomy who treat unknown tags as drafts.

Switch to allow-only mode?

In allow-only mode:

  • Only tags matching at least one enabled rule are shown.
  • All other tags are hidden.
You currently have 3 enabled rules. They will permit roughly 38 tags. The other 162 will be hidden.
Heads up: in allow-only mode, your noise-hiding presets like Hide hex codes act backwards (they would permit hex codes). Consider toggling those presets off, or switching back to default mode where they belong.
